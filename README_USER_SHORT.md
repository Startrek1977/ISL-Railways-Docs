# 🚂 ISL Railways - Quick User Guide

Simple app for managing Israeli railway trains & stations. Perfect for operators & managers!

---

## 🎯 What You Can Do

- ✅ **ADD TRAINS** → Click "Add Train" → Select origin & destination → Click Create
- ✅ **EDIT TRAINS** → Find train → Click Edit → Change route → Click Save
- ✅ **DELETE TRAINS** → Find train → Click Delete → Confirm
- ✅ **VIEW STATIONS** → See all 7 Israeli railway stations
- ✅ **CHANGE LANGUAGE** → 🇺🇸 English | 🇮🇱 Hebrew | 🇷🇺 Russian | 🇺🇦 Ukrainian
- ✅ **FILTER BY DAY** → See trains for specific days

---

## 🚀 Getting Started (5 Steps)

1️⃣ **OPEN** → Go to app URL (administrator provides)
2️⃣ **EXPLORE** → See Stations (left) & Trains (right)
3️⃣ **ADD** → Click "Add Train" → Pick origin & destination → Create
4️⃣ **EDIT** → Find train → Click Edit → Save changes
5️⃣ **DELETE** → Find train → Click Delete → Confirm

## ⚡ Important Rules

🚫 Train numbers: 10-100 only | Must be unique | No duplicates
🚫 Origin ≠ Destination | Pick DIFFERENT stations
🚫 Stations are PERMANENT | Cannot be edited or deleted
