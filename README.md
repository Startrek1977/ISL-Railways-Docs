## 🚀 Getting Started

### What You Need
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection to access the app
- That's it! 🎉

### How to Use (5 Simple Steps)

#### 1️⃣ **Open the App**
```
Navigate to the app URL in your browser
(Your administrator will provide the URL)
```

#### 2️⃣ **See the Dashboard**
You'll see two main sections:
- 📍 **Stations** (left side) - All available stations
- 🚂 **Trains** (right side) - Current trains and routes

#### 3️⃣ **Add a New Train**
- Click the **"➕ Add Train"** button
- Select origin station
- Select destination station
- Click **"Create"**
- ✅ Done! Your new train appears in the list

#### 4️⃣ **Edit a Train**
- Find the train in the list
- Click the **"✏️ Edit"** button
- Change the route
- Click **"Save"**
- ✅ Changes saved!

#### 5️⃣ **Delete a Train**
- Find the train in the list
- Click the **"🗑️ Delete"** button
- Confirm when asked
- ✅ Train removed!

## 🚂 About Trains

### Train Number Rules
- Numbers are between **10 and 100**
- Each train has a **unique number** (no duplicates)
- Cannot edit train number directly

### Station Compatibility
- **Can choose any Israeli station**
- Stations available: Tel Aviv, Haifa, Beer Sheva, Nahariya, and more
- Origin and destination **must be different**
- Cannot select the same station twice

### Train Information You See
| Information | What It Means |
|-------------|--------------|
| 🆔 Train Number | Unique ID for each train (e.g., #25) |
| 🏢 Origin | Where the train starts |
| 🏁 Destination | Where the train ends |

## ℹ️ Frequently Asked Questions

### 🚂 Train Questions
**Q: Can I change a train's number?**
A: No, train numbers are set when created and cannot be changed. You can delete and recreate it if needed.

**Q: Why can't I select the same station twice?**
A: A train must go somewhere! Origin and destination must be different stations.

**Q: What happens if I delete a train?**
A: It's permanently removed. You'll be asked to confirm before deletion.

### 📍 Station Questions
**Q: Can I add new stations?**
A: No, stations are fixed. They represent the permanent Israeli railway network.

**Q: Can I edit station information?**
A: No, station information is permanent and cannot be changed.

### 🌍 Language Questions
**Q: Can I switch languages anytime?**
A: Yes! Just click the language selector at the top. Changes happen instantly.

**Q: Does my language choice get saved?**
A: Yes! The app remembers your language preference.

**Q: Is Hebrew text right-to-left?**
A: Yes! When you select Hebrew, the entire interface automatically adjusts for right-to-left reading.

### 🎨 Design Questions
**Q: Why are the buttons colorful and 3D?**
A: It's a modern design to make the app more enjoyable and visually appealing to use!

**Q: Why is it so fast?**
A: The app is built with modern technology that responds instantly to your actions.

### 💻 Technical Questions
**Q: What browser should I use?**
A: Works great on Chrome, Firefox, Safari, or Edge. Use the latest version for best results!

**Q: Why does my browser need to be modern?**
A: Modern browsers support the latest web technologies that make the app secure and fast.
